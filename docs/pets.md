# Профессии, умения и уровни прокачки питомцев 3b3t  

## Полезные команды

- `/pcst` — выбрать профессию питомца  
- `/petinfo` — показать сведения о питомце  
- `/petname <имя>` — задать имя  
- `/petrelease` — удалить питомца  
- `/petcall` — телепортировать к себе  
- `/petsendaway` — отправить отдыхать  
- `/petrespawn [pay | show | auto]` — возродить питомца  
- `/petstore` — сохранить в предмет  
- `/pettrade <ник> <цена>` — обменяться питомцами  
- `/petshop` — магазин питомцев  
- `/petskill` — список активных навыков  
- `/petstop` — прекратить атаку  
- `/petinventory` — открыть рюкзак питомца  
- `/petpickup` — включить/выключить автоподбор  
- `/petbehavior <режим>` — сменить режим поведения  
- `/petbeacon` — настроить эффект «Маяк»
